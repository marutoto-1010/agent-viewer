GitHub Repo Viewer

GitHubリポジトリの中身をスマホから素早く閲覧

Fine-grained Personal Access Token（Contents: Read権限）

トークンの取得方法

  1. GitHub → Settings → Developer settings
  2. Personal access tokens → Fine-grained tokens
  3. Generate new token
  4. Repository access: Only select repositories → claude-agent
  5. Permissions → Repository permissions → Contents: Read
  6. Generate token → コピーして上に貼り付け
